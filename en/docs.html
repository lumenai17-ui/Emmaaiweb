1. Quick Start

Google Play (Users)

The fastest way to get started.

  • Download from Google Play Store ($2.99 one-time)
  • Open E.M.M.A. and accept permissions
  • Go to Settings and configure your AI provider
  • Open The Forge to create your first agent
  • Say "Hello Emma" to activate hands-free mode

Estimated time: 2 minutes

GitHub (Developers)

For those who want to build, audit or contribute.

# Clone the repository
git clone https://github.com/lumenai17-ui/E.M.M.A-AI.git

# Open in Android Studio
# Requires JDK 17 + Android SDK 35

# Build debug
./gradlew assembleDebug

# Build release (requires keystore)
./gradlew assembleRelease

The GitHub version does not include a pre-compiled APK. You must build from source.

2. Multi-Agent Architecture

E.M.M.A. is an AI agent swarm orchestrator (Agent Swarm Hub) with A2A (Agent-to-Agent) protocol.

ModuleDescription
EmmaEngine (Supervisor)System director. Evaluates and delegates tasks without solving directly, reducing hallucinations
Agent Swarm HubWhatsApp-style interface for multiple agents with independent threads
Koog Engine (A2A)Routing protocol enabling compressed memory handoff between agents
Agent Factory ("The Forge")Dynamic agent creator with System Prompt, icon and brain selection
Compressive HandoffContext transfer between agents under the same thread
Pre-Seed AutoInjects pillar agents ("E.M.M.A. Chat", "Live Vision", "Deep Voice") on first launch
Hermes TunnelPersistent WebSocket (OkHttp) for bidirectional communication with master server

7. Voice & Languages

Voice Engines

  • Deepgram Nova-2 — Ultra low latency STT
  • Deepgram Aura — Premium TTS (English)
  • ElevenLabs — Multilingual voice cloning
  • Android TTS native — Free system fallback

Fallback chain: ElevenLabs → Deepgram → Android TTS

Languages

  • Spanish — UI + Voice + Chat (primary)
  • English — UI + Voice + Chat
  • Others — Adaptive via Locale.getDefault()

Dual XML (i18n): strings.xml in Spanish and English with adaptive translation.
